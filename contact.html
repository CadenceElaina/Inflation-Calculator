This US Inflation Calculator measures the purchasing power of dollar from 1913 to 2022. Enter a start year with an amount in dollars then enter an end year and click "Calculate" below to see how that value changes. The cumulative rate of inflation between the two years will also be displayed.

Hi! Thanks for checking out my Inflation Calculator webpage. I was inspired by bls.gov and in2013dollars.com/us-cpi.The purpose was to hone in my JavaScript skills to work in depth with a large set of data, that was contained in an array objects. Where each year is an object that can be accessed and manipulated. I started by implementing a search function that allows a user to find the searched years data. I plan on adding the ability to enter a year with an amount in dollars that will return the value of that amount in the second year.
